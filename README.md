# Closira AI Assignment

This project is a simple AI-powered customer support workflow created for the Closira AI Engineering Internship assignment.

## Features

- FAQ answering using SOP data
- Lead qualification
- Escalation detection
- Conversation summary generation

---

# Tech Stack

- Python
- OpenAI API
- VS Code

---

# Setup Instructions

## Step 1

Install Python.

## Step 2

Open the project folder in VS Code.

## Step 3

Install dependencies:

pip install -r requirements.txt

## Step 4

Create a `.env` file.

Add:

OPENAI_API_KEY=your_api_key_here

## Step 5

Run the project:

python main.py

---

# Exit Program

Type:

exit

---

# Folder Structure

- main.py → Main workflow
- sop_data.json → SOP information
- logs/ → Escalation logs
- test_transcripts/ → Sample conversation cases
- prompt_design.md → Prompt explanation
